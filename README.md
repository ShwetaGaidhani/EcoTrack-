EcoTrack – Eco-Friendly Lifestyle App

Overview
EcoTrack is a mobile application designed to promote sustainable living by helping users track their daily activities and estimate their carbon footprint. The application provides eco-friendly suggestions and encourages users to adopt responsible habits.

Objectives
To calculate and track the user’s carbon footprint
To suggest eco-friendly daily habits
To promote environmental awareness
To motivate users through rewards and progress tracking

Features
Habit suggestions for sustainable living
Carbon footprint calculation based on user input
Rewards system with points and badges
Dashboard with graphs and progress tracking
Daily reminders for eco-friendly actions

Technologies Used
Android (Java/Kotlin) / Flutter
Firebase (Authentication and Database)
MPAndroidChart (Data Visualization)

System Architecture
User Input → Activity Data → Carbon Calculation → Firebase Database → Dashboard → Rewards

Installation
Clone the repository
Open the project in Android Studio
Connect Firebase
Run the application

Conclusion
EcoTrack is a simple and effective solution for promoting sustainable habits. It helps users understand their environmental impact and encourages eco-friendly behavior.
